# Bird detection

Detect birds in an image file(PNG, JPG, etc) then classify them to another folder.

### Datasets
#### Total
bird: 11
no bird: 1126

#### Test
bird 11
no bird 10

## Performance
14 mins for 1126 images

### YOLOv8

Success: 1 ~ 11
